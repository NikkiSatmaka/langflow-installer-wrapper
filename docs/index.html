Langflow for Windows

Install Langflow 1.9.6 with one click. No Python or package manager needed.

Download Installer
~37 KB · Windows 10 / 11

How to install

1
Download the zip

Click the button above to get the installer.

2
Extract the zip

Right-click and select Extract All, or open it in File Explorer and drag the files out.

3
Double-click Install Langflow.bat

A menu will appear. Press I and hit Enter. The script handles everything.

What the script does

  • Installs uv (a fast Python package manager)
  • Downloads Python 3.12
  • Creates a virtual environment in %USERPROFILE%\langflow\
  • Installs Langflow 1.9.6
  • Creates a desktop shortcut (Langflow.lnk)

After install

Double-click the Langflow desktop shortcut. A terminal window will open. Your browser will launch automatically once the Langflow server is ready at http://127.0.0.1:7860 with Langflow ready.

⚠️ Keep the terminal window open — The Langflow server runs in that terminal. Closing it will stop the server and the browser page will stop working.
